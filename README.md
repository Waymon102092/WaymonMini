# WaymonMini  微信小程序UI组件


#### 介绍
```

  WaymomMini是一个 我自己常用的微信小程序组件库。
  目前有 WaymonVue 和 WaymonMini

```

#### 特性
```
 WaymonBtn

 WaymonButton

 WaymonBottom

 WaymonCheck

 WaymonContent

 WaymonDialog

 WaymonEmpty

 ....

```


#### 快速上手

```

 npm i waymon_mini

````

#### 示例

```
 
构建npm


"usingComponents": {
    "WaymonBtn": "waymon_mini/WaymonBtn/WaymonBtn"
}

<WaymonBtn status="1" title="提交订单" bindtap="onConfirm"></WaymonBtn>
```
